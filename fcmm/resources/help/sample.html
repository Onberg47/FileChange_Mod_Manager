Mod Management View

Overview

This view allows you to manage your game mods. You can add, remove, and organize mods.

Features

  • Drag & Drop: Drag mods from your file system to add them
  • Reordering: Drag and drop to reorder mods
  • Filtering: Use the filter to find specific mods

Usage

To add a mod:

  1. Click the "Add Mod" button
  2. Select a mod file from your computer
  3. Mods will appear in the list

Keyboard Shortcuts

Ctrl + N    - Add new mod
Ctrl + D    - Duplicate selected mod
Ctrl + Delete - Remove selected mod
